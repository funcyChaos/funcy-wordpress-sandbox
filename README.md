# funcy-wordpress-sandbox
A playground to remember all the ways I like to do wordpress things

# Collaborators:

	If you have a feature you'd like to add please follow this standard:
		Make a branch off of main, create your feature there in the most minimal way possible to represent what that feature accomplishes

		No merges into main, all features will be tracked with branches

		If you want to change any of the features/branches create an issue and tag the relevant developer
